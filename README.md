# 📖 Penjelasan Program

## 1️. Array 1 Dimensi (Larik)
Program ini menggunakan **array satu dimensi** untuk menyimpan data mahasiswa seperti:

- Nama
- NIM
- Nilai

Data disimpan dalam array dan ditampilkan menggunakan **perulangan (for)**.

Konsep yang digunakan:
- Array 1 Dimensi
- Tipe data String dan Integer
- Perulangan (for)

---

## 2️. Array 2 Dimensi (Matriks)
Program ini menggunakan **array dua dimensi** untuk menyimpan data mahasiswa dalam bentuk tabel yang terdiri dari:

- Nama
- NIM
- Kelas

Data kemudian ditampilkan menggunakan **perulangan untuk membaca setiap baris array**.

Konsep yang digunakan:
- Array 2 Dimensi
- Matriks
- Perulangan (for-each)

---

## 3️. Program Bangun Ruang
Program ini digunakan untuk menghitung **volume beberapa bangun ruang**, seperti:

- Kubus
- Balok
- Tabung
- Kerucut
- Bola

Program akan meminta input dari pengguna kemudian menghitung **volume berdasarkan rumus matematika bangun ruang**.

Konsep yang digunakan:
- Input menggunakan Scanner
- Percabangan (Switch Case)
- Perhitungan matematika
